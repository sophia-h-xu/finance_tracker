# Finance Tracker
A full-stack finance tracking app that helps users manage income, expenses, and savings with customizable categories and visual insights (pie charts and trends).

# Tech Stack
Frontend: Next.js (React + TypeScript)
Backend: FastAPI (Python)
Database: SQLite
Charts: Recharts

#Backend Setup
cd backend
source venv/bin/activate
uvicorn main:app --reload


#Frontend Setup
cd frontend
npm run dev
